# SkillCraft Technology – Task 01

## Excel Sales Dashboard

This repository contains my submission for **Task 01** of the **SkillCraft Technology Data Analytics Internship**.

## Objective

Build an interactive Excel Sales Dashboard to analyze Superstore Sales Data by:

* Cleaning and organizing the dataset
* Creating Pivot Tables for data analysis
* Building Pivot Charts for visualization
* Visualizing Total Sales
* Analyzing Profit by Category
* Tracking Sales Trends over time
* Applying Conditional Formatting to highlight high and low performers

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting

## Dashboard Features

* Total Sales Overview
* Profit by Category
* Sales Trend Analysis
* Interactive Pivot Table Reports
* Performance Highlights using Conditional Formatting
